# 🚀 IRON RAIDEN

![Gameplay](assets/images/Animation.gif)

A 2D space shooter built with JavaScript that runs directly in the browser.  
Inspired by classic arcade shooter games.

## 🎮 Gameplay

The player controls a spaceship and must destroy enemies while avoiding collisions.

Main mechanics:
- Spaceship movement
- Shooting bullets
- Continuous enemy spawn
- Life system
- Score system
- **Win condition: destroy 20 enemies**

## 🛠 Technologies

- JavaScript
- HTML
- CSS
- DOM Manipulation
